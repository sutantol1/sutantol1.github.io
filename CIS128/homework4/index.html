PWA

A progressive web app (PWA) is an app that's built using web platform technologies, providing installable web application.

Like a website, a PWA can run on multiple platforms and devices from a single codebase. Like a platform-specific app, it can be installed on the device, can operate while offline and in the background, and can integrate with the device and with other installed apps.

Advantages of PWA:

  • Easy to access, no need to install anything
  • Minimal storage requirement
  • Fast, can work offline
  • Offering a high-quality user experience
